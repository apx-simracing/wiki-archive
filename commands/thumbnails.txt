====== thumbnails ======

===== Purpose =====

Downloads the mod thumbnails within a tar.gz file to the local machine

===== Arguments =====

The filename.

===== Examples =====

<code>
apx.py --server servername --cmd thumbnails --args thefile.tar.gz
</code>

===== Remarks =====

Requires running server

<WRAP center round info 60%>
The Id's of the images are not stable. They might differ from client to client.
</WRAP>
